# PetCom

## Tech Stack

### Front end
Flutter

```
# flutter doctor
flutter run
```

Useful tools: 
- [Json_to_dart](https://javiercbk.github.io/json_to_dart/)

### Server
Spring, MyBatis, Maven, Docker...

Under /community folder
```
docker-compose up
```
Related articles:
- [Docker with Spring](https://www.javainuse.com/devOps/docker/docker-mysql)


### Database

Mysql 8.0
